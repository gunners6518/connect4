#プロジェクトの概要
ブラウザ上で遊べるconnect4です。

##使用技術
React(クラスコンポーネント)
TypeScript

##要件
- ゲーム開始前、ゲーム中、ゲーム後の3つの状態がある
- セルをクリックするとその列の一番下に駒が配置されて、相手プレイヤーの番になる
- タテ、ヨコ、ナナメのいずれかに同じ色が4つ連続したら、そのプレイヤーの勝ちになる
